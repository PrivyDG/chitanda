# snowball 

An IRC bot framework. Requires Python 3.6+.

## Setup

```bash
$ git clone git@github.com:dzlr/snowball.git
$ cd snowball 
$ cp config.txt config.py  # Edit config values.
$ python3 -m virtualenv .venv
$ source .venv/bin/activate
$ pip install -e .
```

## Running

```bash
$ cd snowball
$ source .venv/bin/activate
$ snowball
```
